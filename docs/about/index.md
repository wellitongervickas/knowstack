# About KnowStack

> **Note:** This section contains project information, not technical documentation. For technical docs, see [Tutorials](../tutorials/index.md), [Guides](../guides/index.md), [Reference](../reference/index.md), or [Explanation](../explanation/index.md).

KnowStack is an open-source, self-hostable, local-first platform for AI-powered documentation queries. It serves docs, agents, skills, commands, templates, and memory via MCP — working with any MCP-compatible AI tool.

## What is KnowStack?

KnowStack enables development teams to query their documentation using AI. Connect your IDEs and AI tools directly to your living documentation.

**Key capabilities:**

- **AI-Powered Queries** - Ask questions, get answers from your docs
- **Multi-Tenant** - Each organization has isolated data
- **Provider-Agnostic** - Swap AI providers without code changes

## Use Cases

### IDE Integration

Connect your IDE to KnowStack and query documentation without leaving your editor.

### Team Documentation

Centralize your team's documentation and make it searchable with AI.

### Cross-Team Knowledge

Enable frontend developers to query backend API specs and vice versa.

## Architecture

KnowStack is built with:

- **NestJS 11** - Node.js framework
- **TypeScript** - Type-safe development
- **PostgreSQL 16** - Relational database
- **Redis 7** - Caching
- **Clean Architecture** - Maintainable code structure
